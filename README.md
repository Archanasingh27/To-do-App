### React To-Do List Application
### Project Overview

This is a responsive To-Do List application built using React and Vite. The application helps users organize and manage their daily tasks by allowing them to add, edit, delete, and mark tasks as completed. It also includes a dynamic progress bar that updates automatically based on completed tasks.

### Features
Add new tasks
Edit existing tasks
Delete tasks
Mark tasks as completed
Dynamic progress bar based on completed tasks
Task counter showing completed and total tasks
Responsive design for desktop, tablet, and mobile devices
Clean and modern user interface
Built using React Functional Components and Hooks

### Installation

Clone the repository
git clone https://github.com/Archanasingh27/To-do-App.git
Move to the project folder
cd todo-app
Install dependencies
npm install
Start the development server
npm run dev

Open the application in your browser using the local URL shown in the terminal (usually http://localhost:5173).

### How to Use
Enter a task in the input field.
Click the Add button to add the task.
Click the Edit button to modify a task.
Click the Save button after editing.
Click the Delete button to remove a task.
Mark a task as completed using the checkbox.
The progress bar and task counter update automatically as tasks are completed or deleted.
